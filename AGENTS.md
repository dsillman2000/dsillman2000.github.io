# AGENTS.md

## Project

David Sillman's personal website and blog at [www.dsillman.com](https://www.dsillman.com). Static site built with Jekyll 4 and the custom [Swimmer](https://github.com/dsillman2000/swimmer) theme (forked from Poole). Deployed to GitHub Pages via `.github/workflows/github-pages-mzrn.yml`.

## Tech Stack

- **Jekyll 4.4** (Ruby 3.3.2) -- static site generator
- **Tailwind CSS v4** -- all styling, compiled via `@tailwindcss/cli`
- **Kramdown (GFM)** -- Markdown rendering
- **Rouge** -- syntax highlighting (light + dark mode)
- **MathJax 2.7** -- LaTeX math in posts
- **fnm** -- Node.js version manager (Node 24, see `.node-version`)

## Commands

| Command | Description |
|---------|-------------|
| `make install` | Install Ruby gems, npm packages, and download fonts |
| `make serve` | Compile CSS + watch + `jekyll serve --livereload` |
| `make build` | Compile CSS + `jekyll build` |
| `make css` | Compile and minify Tailwind CSS only |
| `make clean` | Remove `_site/` and `.jekyll-cache/` |
| `make resume` | Generate resume `.docx` and `.pdf` into `assets/docs/` |

## File Structure

- `_config.yml` -- Jekyll config; lists plugins and `exclude` paths
- `_css/` -- Tailwind CSS source partials; entry point is `_css/styles.css`
- `_includes/` -- HTML partials: `head.html`, `mathjax-head.html`, `post-sidebar.html`
- `_layouts/` -- Templates: `default.html` (base), `page.html`, `post.html`
- `_posts/` -- Blog posts in Markdown with YAML front matter
- `assets/` -- Static files: `fonts/`, `images/`, `docs/`
- `styles.css` -- Generated Tailwind output (do not edit directly)
- `build_resume.py` -- Python script generating resume with Swimmer Light styling

## Blog Post Conventions

- Filename format: `YYYY-MM-DD-slug.md`
- Front matter fields: `layout: post`, `title`, `related` (list of related post slugs)
- Posts use `post` layout, which includes MathJax support
- Series posts share a slug prefix (e.g., `yamlextras`, `gcpstaticsite`)

## CSS / Styling

- All styles live in `_css/` as Tailwind CSS partials imported by `_css/styles.css`
- Theme colors and fonts defined in `_css/theme.css` via `@theme` and `:root` variables
- Light/dark mode toggle via `data-theme` attribute and `localStorage`
- Do not edit `styles.css` at the project root -- it is generated by `make css`

## Excluded from Jekyll Build

Files listed in `_config.yml` under `exclude:` are not included in `_site/`. This includes `AGENTS.md`, `README.md`, `Makefile`, `_css/`, `node_modules/`, `Gemfile*`, `package*.json`, and `.github/`.
